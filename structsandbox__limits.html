§ dir_perm

enum sandbox_limits::dir_perm : unsigned short

Allowed permissions for directory bindings.

Parameters
RORead only.
RWRead and write access.
NOEXECDisallow execution of binaries.
FSInstead of binding a directory, mount a device-less filesystem.
MAYBESilently ignore the rule if the directory to be bound does not exist.
DEVAllow access to character and block devices.
Warning
Not all options must be supported by all sandboxes. Please, consult your sandbox documentation first.

float sandbox_limits::extra_time = 0

Set extra time before kill the process. If program finishes in this extra amount of time, it won't succeeded, but total run time will be reported to results log. This time is also in (fractional) seconds.
