Detailed Description

template<typename context_t>
class command_context< context_t >

Context for all commands. This class is templated (and inherited) by dependent part of context for every use-case. Commands processed in "main" thread (eq. in broker_connection class) has dependent part of context broker_connection_context, commands in "job" thread (eq. in job_receiver class) has job_client_context as dependent part of context. For more info, see command_holder class.
