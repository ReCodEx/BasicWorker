std::shared_ptr< task_metadata >  task_meta = nullptr 
)
virtual

Create internal task. This could be one of predefined operations like move or copy file, extract archive etc. which don't need sandbox to execute or root task, ie. base holder for all tasks in dependency graph.

Parameters
idIdentifier of newly created task.
task_metaMetadata for newly created task. If this parameter is nullptr or is not specified, root_task is created. Otherwise, type of created task is detected from given metadata.
Returns
Pointer to task's base type holding proper task type. If requested task type is unknown, nullptr is returned.
